# API ve WinForm Nedir?

- **API (Application Programming Interface):** Uygulamaların birbirleriyle veri ve işlem paylaşmasını sağlayan arayüzdür. Genellikle arka planda çalışır ve kullanıcıya görünmez. Örnek: Twitter API, Google Maps API.  

- **WinForm (Windows Forms):** Windows masaüstü uygulamaları için form ve görsel elemanlar oluşturmamızı sağlayan .NET tabanlı bir platformdur. Kullanıcıya görünür ve buton, textbox gibi elemanlarla etkileşim sağlar.  

**Fark:** API arka planda veri/işlem sağlar, WinForm ise kullanıcıya görünür arayüz sunar.
